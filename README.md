## 🤖 AI-Powered Smart Notes Application

An intelligent and modern note-taking application designed to enhance productivity, organization, and workflow management using AI-inspired features and interactive UI experiences.


### 🔗 Live Demo
👉 [Launch App](https://ais-dev-75il7uvdu6oxsnxlyib63z-83373914602.asia-southeast1.run.app/)



### ✨ Core Features

- 📝 Create, edit, and delete notes seamlessly
- 💾 Persistent browser storage using `localStorage`
- 🎨 Responsive clean UI with light & dark mode support
- 📌 Pin important notes to keep them prioritized
- 🔍 Real-time search functionality with highlighted matches
- ↕️ Sort notes by newest or oldest entries
- 🗂️ Card-based dashboard layout for organized viewing
- 📥 Export notes as `.txt` or `.json` files

---

## 🤖 AI & Smart Productivity Features

- 🧠 Automatic note summarization
- 🏷️ Smart tag generation for better organization
- 😊 Mood detection and classification for notes
- 🎤 Voice-to-text note creation using speech transcription
- ✨ AI-assisted structured Markdown formatting
- 📋 Convert notes into interactive task checklists
- 📊 Kanban-style workflow board for task management
